## Support Resources

### If Stuck on Parameter Bug

**Debug Checklist**:
1. Is parameter value actually being received by function? (Add logging in function call handler)
2. Is parameter stored in locals array? (Add logging in SetLocal opcode)
3. Is parameter retrieved correctly? (Add logging in GetLocal opcode)
4. Is upvalue capture interfering? (Check compiler's upvalue logic)

**Questions to Answer**:
- What value is in locals[0] at function entry?
- What does GetLocal return for parameter 0?
- Is it a function object or the argument value?

### If Stuck on Integration Test

**Debug Checklist**:
1. Does test compile? (No syntax errors?)
2. Does compiler run without errors? (Check error messages)
3. Does VM start execution? (Is first opcode reached?)
4. Where does execution fail? (Which opcode?)
5. What error message is shown?

**Approach**:
1. Add println! at opcode handlers
2. Run test with `--nocapture`
3. Follow execution trail
4. Find where it diverges from expected
