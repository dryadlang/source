# projeto-completo

Projeto Dryad criado com Oak.

## Executar

```bash
oak run start
```

ou

```bash
dryad run main.dryad
```

## Scripts Disponíveis

- `oak run start` - Executa o projeto
- `oak run test` - Executa testes
- `oak run check` - Verifica sintaxe

## Dependências

Veja o arquivo `oaklibs.json` para gerenciar dependências.
